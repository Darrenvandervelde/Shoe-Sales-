## Project Overview
This project analyzes a small dataset of shoe sales over a year. The data is stored in an array and includes monthly sales of different shoe types. The analysis calculates key metrics such as total revenue, monthly revenue, and revenue per shoe type, providing insights into sales performance throughout the year.

## Features
- Stores shoe sales data in a structured array format.
- Calculates total yearly revenue.
- Computes monthly revenue for each month.
- Aggregates revenue by shoe type to identify top-selling products.
- Provides key insights based on the sales trends.

## Dataset
The sales data is stored as a list of lists in a Array ( Python ) 

## Analysis Performed

- **Print Sales Data** – Display all sales records month by month.
- **Total Revenue** – Calculate total revenue for the year.
- **Monthly Revenue Summary** – Compute revenue for each month.
- **Revenue by Shoe Type** – Sum revenue per shoe type to identify best-selling products.
- **Key Insights** – Highlight trends such as seasonal performance and high-performing products.

## Key Insights

- Sneakers consistently generated strong sales across multiple months.
- Sandals performed best during summer months.
- Boots peaked toward the end of the year.
- Overall yearly revenue shows consistent demand for shoes.

## How to Use

1. Open the `Shoe Sales Data Analysis.ipynb` notebook in Jupyter Notebook or JupyterLab.
2. Run all cells sequentially to view sales data, calculations, and insights.
3. Modify or expand the `sales_data` array to include new products or months for further analysis.
